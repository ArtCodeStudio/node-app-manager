# node-app-manager
Manage your Node.js applications

# Example

Start the example with:

```sh
yarn run start # or start:prod
```

or to start the app manager itself also with PM2:

```sh
pm2 start ./pm2.example.config.js # or ./pm2.example-prod.config .js
```
